# Permissions — Documents

| Capacité | Description |
|----------|-------------|
| `documents:read` | Lecture selon liens / rôle |
| `documents:upload` | Dépôt |
| `documents:approve` | Valider pièce sensible |
| `documents:delete` | Suppression / archivage |

Scope souvent hérité de l’entité liée (projet, finance).
