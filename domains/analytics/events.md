# Événements — Analytics

Souvent **dérivés** des domaines métier. Événements propres :

| Événement | Description |
|-----------|-------------|
| `Dashboard.Refreshed` | *(technique)* |
| `AlertRule.Triggered` | Seuil franchi |
| `Insight.Generated` | Sortie IA / scoring |

Les KPI ne **créent** pas d’événements métier ; ils **réagissent** aux événements des autres domaines.
