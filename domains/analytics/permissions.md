# Permissions — Analytics

| Capacité | Description |
|----------|-------------|
| `analytics:dashboard:read` | Voir tableaux de bord |
| `analytics:dashboard:configure` | Modifier widgets / partages |
| `analytics:export` | Exporter données agrégées |
| `analytics:admin` | Règles d’alerte globales |

Respecter la **sensibilité** des données agrégées (finance, RH).
