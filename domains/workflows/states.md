# États — Workflows

## Traitement d’événement

| État | Description |
|------|-------------|
| `received` | Reçu |
| `processing` | En cours |
| `completed` | Succès |
| `failed` | Erreur |
| `dead_letter` | Abandon contrôlé |

## Règle

| État | Description |
|------|-------------|
| `draft` | Non publiée |
| `active` | Exécutée |
| `paused` | Suspendue |
| `deprecated` | Remplacée |
