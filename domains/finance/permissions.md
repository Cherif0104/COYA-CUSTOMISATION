# Permissions — Finance

| Capacité | Description |
|----------|-------------|
| `finance:journal:read` | Lecture écritures |
| `finance:journal:draft` | Création brouillon |
| `finance:journal:approve` | Validation |
| `finance:journal:post` | Comptabilisation |
| `finance:invoice:*` | Cycle facturation |
| `finance:report:read` | Rapports |

Scopes : `organization`, éventuellement `project` pour imputation.
