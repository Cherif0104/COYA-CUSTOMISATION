# Permissions — Workflows

| Capacité | Description |
|----------|-------------|
| `workflows:rules:read` | Consulter règles |
| `workflows:rules:write` | Publier / modifier règles |
| `workflows:replay` | Rejeu contrôlé (admin) |
| `workflows:dlq:manage` | Gestion file morte |

Typiquement **réservé** admin org / super admin.
